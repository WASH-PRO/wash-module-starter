Стартер модуля WASH PRO CRM

Шаблон для разработки собственных расширений. Daemon пишет heartbeat в data/last_snapshot.json.

Структура репозитория

  • wash-module.json — манифест для каталога CRM
  • src/main.py — процесс в PyOrchestrator
  • ui/index.html — настройки в Dashboard
  • ui/help.html — этот файл справки

Публикация

  1. Fork / clone этого репозитория
  2. Измените id в манифесте
  3. Добавьте запись в modules/registry.json CRM
