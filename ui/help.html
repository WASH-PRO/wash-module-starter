WASH PRO CRM Module Starter

Template for building custom extensions. The daemon writes heartbeat to data/last_snapshot.json.

Repository layout

  • wash-module.json — manifest for the CRM catalog
  • src/main.py — PyOrchestrator process
  • ui/index.html — settings in Dashboard
  • ui/help.html / ui/help.ru.html — help (English / Russian)

Publishing

  1. Fork / clone this repository
  2. Change id in the manifest
  3. Add an entry to CRM modules/registry.json
